# 🩺 Diabetes Prediction Web Application

This is a machine learning-based web application that predicts the risk of diabetes using user health parameters.

---

## 🚀 Features

- Predicts diabetes risk (Low Risk / Prediabetic / High Risk)
- Dynamic UI with color-based risk indication
- Interactive progress bar showing probability
- Personalized diet and exercise recommendations
- Clean and user-friendly interface

---

## 🛠 Tech Stack

- Python
- Flask
- Machine Learning (Logistic Regression)
- HTML, CSS

---

## 📂 How to Run

1. Clone the repository:
   git clone https://github.com/ashwitharooth-9119/diabetes-prediction-web-app

2. Install dependencies:
   pip install flask numpy pandas scikit-learn

3. Run the application:
   python app.py

4. Open in browser:
   http://127.0.0.1:5000
